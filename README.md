Playwright Assignment 1 (Python)

 Project Description

This project contains UI test automation scripts developed using Playwright with Python.

Requirements
Make sure the following are installed on your system:
                                        Python 3.11 or higher
                                        pip (Python package manager)


Install Dependencies

Run the following commands in the project folder:
                                        pip install -r requirements.txt
                                        playwright install


 Run Tests

To execute the test scripts, run:
                                python test_script.py
(Replace test_script.py with your actual test file name if different)



Project Structure
tests/ → contains test scripts
requirements.txt → list of dependencies
README.md → project instructions
